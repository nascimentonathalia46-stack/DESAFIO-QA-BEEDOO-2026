# DESAFIO-QA-BEEDOO-2026
Análise de teste QA

## Objetivo da aplicação

Pelo que foi possível observar ao explorar a aplicação, o objetivo principal do sistema é permitir o gerenciamento de cursos dentro de uma plataforma. A aplicação possibilita cadastrar novos cursos e visualizar os cursos que já foram registrados.

A ideia parece ser facilitar a organização de cursos em um único lugar, permitindo que os usuários possam consultar e administrar essas informações de forma simples.

## Principais fluxos disponíveis

Durante a navegação na aplicação, foi possível identificar alguns fluxos principais do sistema:

- Cadastro de curso
- Listagem de cursos
- Visualização das informações do curso
- Navegação entre as telas

## Pontos mais críticos para teste

Alguns pontos da aplicação chamaram mais atenção durante a análise e considero que são áreas importantes para serem testadas:

- Cadastro de cursos
- Validação dos campos
- Listagem de cursos
- Persistência das informações
- Testes com comportamentos inesperados
